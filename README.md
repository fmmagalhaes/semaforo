# Semáforo

A simple implementation of "Jogo do Semáforo" (Alan Parr's Traffic Lights).

## Play

Visit https://fmmagalhaes.github.io/semaforo/

## AI Disclaimer

Built with AI (Claude Opus 4.7). The first and second commits were generated using the following prompts:

```
Build a minimalist and modern implementation of "Jogo do Semáforo" (Alan Parr's Traffic Lights) using only vanilla HTML, CSS, and JavaScript.

Requirements:

- Single page application only
- The game starts immediately when the page loads
- No setup screen or menus
- Minimalist UI with clean spacing and large readable elements
- Responsive design that works well on desktop, tablet, and mobile
- Use Flexbox for all layouts
- No frameworks or libraries

Open in the browser when you're done
```

The prompt above yielded a fully working game in under than 1 minute.

The following prompt was then used to improve code readability:

```
Separate js, html and css in different files; Improve spacing for code readability
```

Which again took less than 1 minute.

Further improvements will not be documented.

### Alternative Prompt

A much shorter prompt such as "Implement Alan Parr's Traffic Lights (3x4)" should also yield a fully functional game. The extra detail mainly shapes the aesthetic and code organisation.
